# express-API
API built with ExpressJS

## Run the API locally
	1. open cmd in the API directory
	2. $nodemon index.js
	
*runs on port 3000 by default*

creating an environmental variable for production
to change port in cmd 

		$set PORT=5000 (for windows)
		$nodemon index.js
		$Listening on port 5000...

## Description

test API for tasks (used by React Application)
3 tasks are set by **default**
